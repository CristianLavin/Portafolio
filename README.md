# Portafolio Personal

Este proyecto es un portafolio web personal desarrollado para presentar el perfil profesional de Cristian Lavin como desarrollador Full Stack. La aplicación muestra una introducción personal, habilidades técnicas, enlaces de contacto y una sección de proyectos con miniaturas, reseñas y una descripción breve de la arquitectura utilizada en cada caso.

## Características

- Interfaz moderna y responsive.
- Cambio de idioma entre español e inglés.
- Sección "Sobre Mí" con stack y habilidades técnicas.
- Sección de proyectos con miniaturas, reseña y arquitectura resumida.
- Enlaces directos a GitHub, LinkedIn y WhatsApp.

## Tecnologías

- React
- Vite
- Tailwind CSS
- Lucide React
- React Icons

## Estructura del Portafolio

- `Hero`: presentación principal con nombre, rol y accesos de contacto.
- `Sobre Mí`: descripción profesional y tecnologías dominadas.
- `Proyectos`: tarjetas con trabajos destacados y su stack técnico.
- `Footer`: cierre simple con derechos reservados.

## Ejecución local

```bash
npm install
npm run dev
```

## Objetivo

El objetivo de este portafolio es servir como carta de presentación profesional, mostrando experiencia, tecnologías dominadas y ejemplos de proyectos desarrollados de forma clara, visual y accesible.
